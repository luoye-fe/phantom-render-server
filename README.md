phantom render server
========================

`http://api.luoyefe.com/phantom/render`

`get`

### options

* url

	render url

* html

	render html code (`url` first)

* width

	viewportSize width

* height
	
	viewportSize height	

* format

	file format, `pdf` `png` `jpeg` `bmp` `ppm`

* quality

	0-100
